# SCT_DS_2 - Titanic Dataset EDA

## Objective
Perform data cleaning and exploratory data analysis (EDA) on the Titanic dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
- Data Cleaning
- Missing Value Handling
- Survival Analysis
- Gender vs Survival Analysis
- Age Distribution Analysis
- Correlation Heatmap

## Outputs
- Graphs and visualizations
- Data insights and trends
